## How to Use

1. **Open a podcast episode** - Tap on any episode to start playback
2. **Tap the microphone button** - Located below the playback controls
3. **Speak your command** - The mic will glow when listening
4. **See confirmation** - A success message appears when command is recognized

## Troubleshooting

**Voice commands not working?**
1. Check microphone permissions in Settings
2. Ensure you have internet connection (for speech recognition)
3. Make sure you're speaking clearly
4. Try tapping the mic button again

**Commands not recognized?**
- Try using the exact phrases listed above
- Speak more slowly and clearly
- Check if your device supports speech recognition
